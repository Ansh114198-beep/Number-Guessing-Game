# Number-Guessing-Game

Simple console number guessing game written in C++.

Build and run
-
Compile with g++ (C++17):

```
g++ -std=c++17 -O2 main.cpp -o guess
```

Then run the program:

```
./guess
```

Gameplay
-
The program chooses a random number between 1 and 100. Enter integer guesses; the program responds with "Too low", "Too high", or "Correct" and shows attempt count.
